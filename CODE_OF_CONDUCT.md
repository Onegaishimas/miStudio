# Code of Conduct

## Our Standards

miStudio is a research tool for mechanistic interpretability. We want participation in this project to be a positive experience for everyone.

Expected behavior:
- Be respectful and constructive in discussions
- Focus feedback on the work, not the person
- Assume good intent and ask for clarification before criticism

Unacceptable behavior:
- Harassment, insults, or personal attacks
- Spam or off-topic promotion

## Enforcement

Violations can be reported to the maintainers via GitHub. Reports will be reviewed and addressed at maintainer discretion.

This project applies the [Contributor Covenant](https://www.contributor-covenant.org/) Code of Conduct v2.1 as its baseline standard.
